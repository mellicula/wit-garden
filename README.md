# WIT Garden 🌱

A living site built by the Women in Tech club — members past and present.

**[garden](index.html)** — member drawings and bios
**[the board](resources.html)** — community-curated links, songs, recipes, and more

---

## Want to contribute?

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full step-by-step guide.

The short version:
1. Fork → clone → branch
2. Add your drawing to `assets/members/`
3. Add your entry to `data/members.js`
4. Add a recommendation to `data/resources.js`
5. Commit → push → open a PR

---

## Tech

Plain HTML, CSS, and vanilla JS. No build step, no dependencies.
Hosted on GitHub Pages.

To run locally just open `index.html` in a browser.
